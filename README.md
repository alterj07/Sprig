# Sprig
For Hack Club's Sprig
Created by using Sprig

A simple mind-stretching single player game to warm-up your mind!
Consume as many pieces of cheese as you can!
Dodge enemies!
Memorize patterns!

W - Move Up
A - Move Left
S - Move Down
D - Move Right
